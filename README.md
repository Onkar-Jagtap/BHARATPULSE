# BharatPulse AI - Political Intelligence Platform

BharatPulse AI is a revolutionary political intelligence and constituency analytics platform designed for the complex landscape of Indian elections. It leverages historical data and real-time survey results to provide strategic insights powered by Gemini 1.5 Flash.

## 🚀 Key Features

*   **War Room Dashboard**: Real-time KPI monitoring including voter satisfaction, volatility, and district coverage.
*   **Survey Hub**: Seamless ingestion of constituency survey data (CSV/XLSX) with AI-powered cleaning.
*   **AI Strategy Intelligence**: Strategic action plans generated by Gemini AI based on voter sentiment and demographic shifts.
*   **Geo-Intelligence**: Geospatial visualization of voter concentration and party strongholds.
*   **Swing Analyzer**: Advanced drift detection algorithms to identify undecided voters and high-risk zones.
*   **Neural Chat**: Interactive AI assistant for deep-diving into constituency datasets.

## 🛠 Tech Stack

*   **Frontend**: React 19, TypeScript, Vite, Tailwind CSS 4.
*   **Animations**: Motion (Framer Motion).
*   **AI Engine**: Google Gemini 1.5 Flash.
*   **Charts**: Recharts.
*   **Icons**: Lucide React.

## 📦 Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Configure your environment: Add `GEMINI_API_KEY` to your secrets/environment.
4.  Run the development server: `npm run dev`.

## 🛡 Security & Privacy

Built with an "Intelligence-First" approach, ensuring all data processing is localized within your secure session. Strategic insights are generated using context-aware AI models without permanent storage of sensitive voter identifiers.
